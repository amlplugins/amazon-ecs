# @amlplugins/amazon-ecs

Native `@aws-sdk/client-ecs` SDK re-exported under the `@amlplugins` namespace.

> Amazon ECS — clusters, task definitions, services, tasks, capacity providers (Fargate/EC2).

## Install

```bash
npm install @amlplugins/amazon-ecs
```

## Use

```ts
import { /* named exports from @aws-sdk/client-ecs */ } from "@amlplugins/amazon-ecs";
// or
import sdk from "@amlplugins/amazon-ecs";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
